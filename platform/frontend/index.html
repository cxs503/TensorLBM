Quick Start
  1. Go to Pre-processing to generate geometry masks or convert physical units to LBM units.
  2. Go to Solve, pick a simulation type, configure parameters, and click Submit Job.
  3. Monitor progress in the Jobs sidebar (left).
  4. Once complete, go to Post-process to view snapshots, plots, and metadata.
  5. Use Benchmarks to run validation suites against published reference data.
